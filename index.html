How does weather vary with Latitude?

This is a mock project built to practice Python requests, APIs and JSON traversals during Data Visualization and Analytics Bootcamp at the University of Minnesota, St. Paul.

There were three main objectives:-

  1. Analyze the relationship between various weather parameters (maximum temperature, humidity, cloudiness and wind-speed) with Latitude.

  2. Visualize the weather of 500+ cities using OpenWeatherMap API using gmap heatmaps

  3. Search for ideal vacation spots (based on certain weather conditions) using Google Places API.

Main observed trends

  • The maximum temperature decreases as distance from the equator increases. This is expected since South and North poles are expected to be very cold due to receiving minimum amount of light and heat from the sun.
  • In general, there are no observable correlation between Humidity and Cloudiness vs Latitude.
  • The maximum temperature of cities in the Northern hemisphere show a strong negative correlation with Latitude as compared to cities in Southern hemisphere, which shows a weak correlation. A possible reasoning is that there are far less habitable locations close to the South Pole.
